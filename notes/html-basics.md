# HTML 学习笔记

## HTML 基础

### 文档结构
```html
<!DOCTYPE html>
<html>
<head>
    <title>页面标题</title>
</head>
<body>
    <!-- 内容 -->
</body>
</html>
```

### 常用标签

#### 文本标签
- `<h1>` - `<h6>`: 标题
- `<p>`: 段落
- `<span>`: 行内容器
- `<div>`: 块级容器

#### 列表
- `<ul>`: 无序列表
- `<ol>`: 有序列表
- `<li>`: 列表项

#### 链接与媒体
- `<a href="">`: 超链接
- `<img src="" alt="">`: 图片
- `<video>`: 视频

#### 表单
- `<form>`: 表单容器
- `<input>`: 输入框
- `<button>`: 按钮
